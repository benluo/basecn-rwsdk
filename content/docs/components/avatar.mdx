## Migrating from Radix UI

<Steps>
  <Step>
    <include>migration-steps/common.mdx</include>
  </Step>
  <Step>
    ### AvatarFallback Delay

    In Radix UI, you can provide a `delayMs` prop to delay its rendering so it only renders for those with slower connections.

    ```tsx
    import { Avatar, AvatarFallback } from "@/components/ui/avatar"

    <Avatar>
      <AvatarImage src="https://github.com/shadcn.png" alt="@shadcn" />
      // [!code highlight]
      <AvatarFallback delayMs={1000}>CN</AvatarFallback>
    </Avatar>
    ```

    With Base UI, you can pass the `delay` prop to achieve the same effect.

    ```tsx
    import { Avatar, AvatarFallback } from "@/components/ui/avatar"
    
    <Avatar>
      <AvatarImage src="https://github.com/shadcn.png" alt="@shadcn" />
      // [!code highlight]
      <AvatarFallback delay={1000}>CN</AvatarFallback>
    </Avatar>
    ```
  </Step>
</Steps>
